Paperlytic is an automated, real time feed designed to track and aggregate the latest academic research papers.

How the system works:

  • The Data Source: The system constantly pulls raw data directly from the Crossref API, which acts as a global database for newly published academic work.
  • The Engine: A serverless Google Apps Script runs autonomously every hour. It fetches new papers, filters out duplicates, formats the data, and ensures everything is clean before pushing it forward.
  • The Brain: The processed data is stored in Supabase a fast PostgreSQL database. To keep the system lightweight and efficient, an automated background job (pg_cron) runs daily to delete old data once storage limits are reached.
  • The Interface: Hosted on Netlify, this frontend connects directly to the database without a middleman server. The result is a minimalist, high speed user interface with live search capabilities and direct DOI links to the original papers.

No clutter. No unnecessary steps. Just a clean pipeline bringing the latest research papers straight to your screen.
